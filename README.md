# AR-Hololens-recogition-and-loctaion-Head
Hololens internal head recognition
An approach and system for real-time registration of patient head CT images and augmented reality overlay display based on a mixed reality headset are disclosed. This method synchronously collects visible light camera images and depth camera images through the ResearchMode interface built into Microsoft HoloLens 2, and obtains the external parameter matrices of the two cameras; the image data is encapsulated using a custom binary protocol and transmitted in real time to the local computing terminal via TCP streaming sockets; the computing terminal detects the head region of the visible light image, aligns the RGB image pixel pairs to the depth map coordinate system (320×288) using the external parameter matrices of the two cameras, applies semantic mask to cut the head depth map in the unified coordinate space, and generates a color 3D point cloud through inverse projection using the internal parameter matrices of the depth camera; after noise reduction and outlier elimination, it is registered with the preoperative CT point cloud to obtain the registration transformation matrix; finally, the registration result is returned to the headset and displayed in real time in the augmented reality field to overlay the internal structure of the patient's head CT. This invention does not require additional hardware calibration equipment. The system is lightweight and has strong real-time performance, and can be used for intraoperative anatomical navigation and medical teaching demonstrations. 
Key words: Mixed Reality; Augmented Reality; HoloLens; Point Cloud Registration; CT Navigation; Depth Camera; ResearchMode; RGB-Depth Alignment
